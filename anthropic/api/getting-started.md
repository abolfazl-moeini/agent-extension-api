# API overview

The Claude API is a RESTful API at `https://api.anthropic.com` that provides programmatic access to Claude models and Claude Managed Agents.

## Prerequisites

- A Claude Console account
- An API key, or Workload Identity Federation

## Available APIs

**General Availability:**
- Messages API
- Message Batches API
- Token Counting API
- Models API

**Beta:**
- Files API
- Skills API
- Agents API
- Sessions API
- Environments API

## Authentication

Headers: x-api-key or Authorization Bearer, anthropic-version, content-type.
